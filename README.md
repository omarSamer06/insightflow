# 🚀 InsightFlow – AI-Powered SaaS Analytics Dashboard

InsightFlow is a full-stack multi-tenant SaaS analytics platform that enables users to manage, visualize, and gain insights from their business data in secure, isolated workspaces.

Built with the MERN stack, InsightFlow combines modern UI, scalable architecture, and intelligent features such as AI-style insights, forecasting, and natural language data queries.

---

## ✨ Features

### 🔐 Authentication & Authorization
- JWT-based authentication
- Role-based access control (Admin / Member)
- Secure protected routes

### 🏢 Multi-Tenant Architecture
- Isolated workspaces per user
- Secure data separation
- Team collaboration with member roles

### 📊 Analytics Dashboard
- KPI cards (totals, trends)
- Interactive charts (line, bar, pie)
- Real-time updates based on user data

### 📁 Data Management
- Full CRUD for records
- Search, filtering, and pagination
- Category-based organization

### 🤖 AI-Powered Features
- **Insights** – Smart summaries of performance
- **Natural Language Queries** – Ask questions like:
  - “Show my highest revenue month”
- **Forecasting** – Predict future trends based on historical data
- **Report Generator** – Structured business reports with recommendations

### 👥 Team Collaboration
- Invite users to workspace
- Role-based permissions
- Shared data access within workspace

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router
- Recharts

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- REST API Architecture

### Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas

---

## 🧠 AI Integration

InsightFlow supports AI-powered features with optional OpenAI integration.

- If API key is provided → dynamic AI responses
- If not → intelligent fallback logic ensures full functionality

This makes the system robust and production-ready even without external AI dependency.
